# User Explanations

Explanations within Informfully are available as tags below item recommendations.

![img/explanation_screen.png](img/explanation_screen.png)

1. Tags for highlighted items.
2. Tags for regular items.

Note that tags can be defined separately for each user-item pair.

 Please see [experiments`, `explanations`, and `explanationViews` in the `Database Documentation](https://github.com/Informfully/Database) on how to set up explanations for your experiment.
